# $psb_ui_chart

Plan-fact charts

## MAM usage

```tree
$psb_ui_chart
	series_x <= years /number
	series_fact <= fact /number
	series_plan <= plan /number
```

## Angular usage

```
<psb-ui-chart
	series_x="[2018,2019,2020,2021]"
	series_fact="[0,100,250,800]"
	series_plan="[50,80,150,500]"
></psb-ui-chart>
```
